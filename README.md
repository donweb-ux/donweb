# DonWeb Message Text Editor

A simple and interactive web-based message editor that allows users to register, log in, compose messages, and generate realistic bank-style receipt popups. This project is built using **HTML, CSS, and JavaScript**, and works fully on the frontend.

## ğŸ› ï¸ How It Works

1. User registers or logs in.
2. User enters message details such as:

   * Bank Title (e.g. Access Bank)
   * Transaction Amount
   * Sender Name
   * Receiver Account Number
   * Receiver Phone Number
3. The system simulates auto-fetching the account name.
4. On **Send**, a success alert appears.
5. A **realistic transaction receipt popup** is displayed.
